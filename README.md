# hello-rust

只是一个学习时简单的 demo 集合

### 基本使用

1. Vscode 安装 `Code Runner`，扩展 ID `formulahendry.code-runner`
2. 代码编辑区域点击鼠标右键，执行 `Run Code`

> Windows 需要使用 WSL 环境

> 若提示无权限，终端执行 chmod +x ./Run.sh
